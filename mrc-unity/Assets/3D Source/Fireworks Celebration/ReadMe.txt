-------------------------
Fireworks Celebration
-------------------------

This asset package not only just provides some firework effects, but also allows you to make a really fireworks celebration! Please watch the demo video to learn what the fireworks celebration made with it looks like. There is no post-processing in the video. The scene file for the demo video is included in the asset package. 

Contains:
- 157 categorized firework prefabs.
- 1 fireworks celebration demo scene.

* Please visit https://youtu.be/tRdKcHfvdno to watch the tutorial  video on setting up light and Haze.
* Because there are too many firework effects in the package, not all firework effects are shown in the demo video,
